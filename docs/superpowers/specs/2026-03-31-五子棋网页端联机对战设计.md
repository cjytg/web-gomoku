# 五子棋网页端联机对战设计文档

## 一、项目概述
将现有五子棋项目从Python Tkinter桌面端迁移到网页端，在保留原有人机对战功能的基础上，增加双人远程联机对战功能。

## 二、需求确认
1. ✅ 保留原有人机对战功能（简单、中等、困难三个难度
2. ✅ 双人联机采用房间号模式，无需注册账号
3. ✅ 响应式设计，支持移动端访问
4. ✅ 开发和部署成本为0，使用免费服务
5. ✅ 界面美观，联机低延迟

## 三、技术选型
- **前端框架**: React 18 + TypeScript + Tailwind CSS 3
- **实时通信**: Supabase Realtime
- **部署**: Vercel (前端) + Supabase (后端服务)
- **总成本**: 0元/月

## 四、系统架构
```
[前端A] ↔ [Supabase Realtime] ↔ [前端B]
            ↓
      [实时数据库]
```
- 无自有后端服务，所有逻辑在前端实现
- AI算法运行在客户端，无需服务器算力
- 实时同步通过Supabase提供的WebSocket服务实现

## 五、核心功能模块

### 1. 游戏核心模块
- **棋盘逻辑**: 15x15棋盘，胜负判断逻辑完全复用现有Python版本
- **AI算法**: 将现有Python版的启发式评分算法重写为JavaScript版本
- **游戏状态管理**:
  - 落子逻辑
  - 悔棋功能
  - 胜负检测
  - 最后落子标记
- **难度系统**: 保留简单、中等、困难三个难度等级

### 2. 联机对战模块
- **房间系统**:
  - 创建房间：生成6位数字房间号
  - 加入房间：输入房间号即可加入对战
  - 房间状态管理：等待中、游戏中、已结束
- **实时同步**:
  - 落子实时同步
  - 游戏状态同步
  - 玩家上下线状态同步
- **断线重连**: 刷新页面后自动恢复游戏状态

### 3. UI模块
- **界面设计**:
  - 保留原有木色棋盘的视觉风格
  - 现代简洁的UI设计
  - 响应式布局，适配PC和移动端
- **交互**:
  - PC端：鼠标点击落子
  - 移动端：触摸点击落子
  - 动画效果：落子动画、胜利动画
- **页面结构**:
  - 首页：选择游戏模式（人机、本地双人、联机对战
  - 游戏页：棋盘、操作按钮、状态提示
  - 房间页：创建/加入房间界面
